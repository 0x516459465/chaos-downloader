# Chaos Downloader

This tools is designed to deal with chaos api from projectdiscovery.io

# Installation

```
git clone https://github.com/ali-0x11/chaos-downloader/
cd chaos-downloader
pip3 install -r requirements.txt
```

# Usage

```
chmod +x ./chaos-downloader.py
./chaos-downloader.py
```
